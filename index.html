📋 Formato CSV esperado (separado por ; )
nome;cpf;rg;naturalidade;nascimento;mae;pai;estadocivil;profissao;pele;altura;peso;email;apelidos;ip;vpi;delegado;agente;obs
• Primeira linha = cabecalho (obrigatorio)  • Apelidos separados por |  • Salve o Excel como CSV UTF-8
Formato JSON (array de objetos)
[ {"nome":"JOAO SILVA","cpf":"123.456.789-00","ip":"2026-001","apelidos":"Joao|Ze"} ]
Aceita exportacoes do proprio SIPOL (botao Exportar CSV convertido para JSON)

NOME;CPF;NATURALIDADE;NASCIMENTO;IP/PROCEDIMENTO
Apenas Nome e obrigatorio. Os demais campos sao opcionais.

💡 Como preparar no Excel / Google Sheets
1. Crie uma planilha com os cabecalhos exatamente como mostrado acima (linha 1)
2. Preencha os dados a partir da linha 2, um investigado por linha
3. Salve como: Arquivo → Salvar Como → CSV UTF-8 (delimitado por virgulas)
4. Se usar Excel, o separador pode ser ; ou , — o sistema detecta automaticamente
5. Campos vazios podem ser deixados em branco (nao precisa preencher todos)
